# LMW Chart Pattern Detector + CNN

Companion demo for the MIT 15.C51 Spring 2026 final project: 
"Chart Patterns, Convolutional Networks, and the Question of Predictive Power."

Replicates Lo, Mamaysky & Wang (2000) classical chart-pattern detection 
on live ticker data, plus inference from a JKX-style CNN trained on 
2004-2017 CRSP data.

## Running locally
pip install -r requirements.txt
streamlit run demo_app.py
